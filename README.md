### ttsTools 合成tts

#### 百度

1. tts合成 https://cloud.baidu.com/product/speech/tts_online  
- 将 Token.appKey 和 Token.secretKey 替换成自己在百度平台申请的appKey/secretKey,
然后直接运行Main.main()即可。  
- 如果想要修改声优，修改TtsOnline.per属性，可选的属性值在 TtsOnline.voiceMap 中
